This is a website dedicated to making a random password that is between 8-128 characters long. Start by clicking the red generate password button the you can select if you want uppercase letters, lowercase letters, special characters, or numbers for your password. after you select what you want in your password it then makes the password for you in the center box that you can then copy and then paste.
